# Galgje in PHP

---

## Wat hebben wij gemaakt ?

Wij twee ( Haroun en Luc ) hebben samen een website gemaakt om Galgje te spelen.

## Welke techonologieen hebben wij hier voor gebruikt?

1. **PHP**
2. **HTML**
3. **CSS**

## Hoe run je dit project ?

Het project run je zo :

1. Download XAMPP
2. Clone het project: `git clone git@github.com:LucBuigel15/galgje-in-PHP.git`
3. Zet het galgje.php bestand in de htdocs folder van XAMPP
4. Start XAMPP op.
5. De website opent automatisch.
